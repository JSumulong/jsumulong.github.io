Projects

Totem

Project Totem is a native Android application that allows users to find other users in augmented reality. This project was designed to allow the contributing members an opportunity to learn new technology at a rapid pace under the pressure of an eight day time constraint and with the expectation of presenting a working demo during the 2016 Coyotes graduation ceremony. My responsibility was to learn enough about the Android Studio dev environment, Java, and XML to create the necessary native views and behaviors for the app. I also paired with another student dev to research and develop the client side web socket necessary for the app.

The front end of Project Totem requires a connection to the web socket server, which can be viewed here. The web socket server is the back end of the project and has recently been deployed on Heroku using Node.js.

Remindr

This is a web app that allows users to store a list of contacts and schedule reminders to be sent to a contact or a group of contacts on one's contact list via text messages. The reminders may be recurring and users can specify the number of times and frequency at which the reminder can be sent.

Remindr was built over a two and a half day sprint according to the specifications of an instructor playing the role of a client. I was responsible for building the user model and controller, applying BCrypt password encryption, developing the contacts controller and views, and implementing the Heroku Scheduler to scrape the database for current reminders to be sent.

Click here to view a deployed version of the project.

NOTE: Texting through Twilio and the Heroku Scheduler on the deployed version have been turned off in order to refrain from accruing charges.

Tutorials

Here is a list of tutorials that I have worked through and found helpful:

  • JavaScript30 (Free)

    • If you're going to be a web developer, you've gotta learn JavaScript. With so many JavaScript tools, libraries, and frameworks floating around it's easy to get overwhelmed. But before you pull your hair out trying to keep up with the latest and greatest that the JavaScript community has to offer, you should learn some plain ol vanilla JavaScript

  • Complete Introduction to React (Paid)

    • React is pretty popular and if you're not familiar with it, this is a solid intro. It's not for the feint of heart, though. Implicit in the 'complete' introduction to React is a handful of pretty popular tools that are often paired with React such as: Redux, JSX, Babel, Webpack, React Router, and some tools for testing as well. It ain't free, but it's a pretty meaty course and you'll most likely find yourself having to review the material just to take it all in. Be warned!

  • Flexbox Froggy! (Free)

    • CSS can be fun. Don't believe me? Try this quick flexbox tutorial and bring the froggies home while picking up a new skill or flexing some CSS muscle.

  • FunFunFun Recursion (Free)

    • Recursion can be confusing. This is a great video that explains the basics of how to approach thinking recursively and how it might be useful when solving problems. Oh yeah, and it's short. Take 15 minutes and invest in thinking recursively. Also, this video is part of a series that I would also recommend about functional programming.

Reading

Articles, blogs, and books - here are a couple of worthwhile reads:

  • To Agility and Beyond: The History - and Legacy - of Agile Development

    • Agile development is almost ubiquitous these days. Read about early development methodologies and how the dominant paradigm has shifted in the relatively short history of computer programming.

    • This is a great introduction to Object Oriented Design and because it's written in Ruby, it's even beautiful to read and easy to follow along. This book takes a practical approach to OO, so it focuses more on the how, than the why.
